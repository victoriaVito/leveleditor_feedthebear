Feed the Bear — Reestructuración de Confluence

Código listo para pegar en el editor de código fuente de Confluence Server

Instrucciones paso a paso

  1. Abre la página en Confluence → Click Edit (o tecla E)
  2. Abre el editor de código fuente: en la toolbar, click el icono <> (Source editor) o usa el atajo Ctrl+Shift+D
  3. Selecciona TODO el contenido actual (Ctrl+A)
  4. Pega el código del Bloque A (abajo) — esto reemplaza toda la página
  5. Adapta el contenido: reemplaza los placeholders [TU CONTENIDO AQUÍ] con tu texto real. Puedes copiar secciones del código fuente original antes de reemplazar
  6. Guarda con Ctrl+S o click en Save

⚠️ Consejo importante: Antes de reemplazar, copia el código fuente actual en un archivo de texto como backup. Así puedes volver atrás si algo no queda bien.

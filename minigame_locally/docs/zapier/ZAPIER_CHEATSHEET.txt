ZAPIER ZAP SETUP (create these in zapier.com)

  2. Task Router
     Trigger: Webhooks by Zapier
     Then: Send to Slack, Google Sheets
     
  3. Learning Pipeline
     Trigger: Webhooks by Zapier
     Then: Send to Claude, Codex for ML updates
     
  4. Doc Publisher
     Trigger: Webhooks by Zapier
     Then: POST to Confluence API
     
  5. Agent Handoffs
     Trigger: Webhooks by Zapier
     Then: Route to next agent

EXAMPLE WORKFLOW
  1. Edit docs/PROJECT_MASTER_HANDOFF.md
     └→ watcher detects change
     └→ sends to Zapier
     └→ Zapier notifies agents
     └→ Claude analyzes, Codex implements, Copilot validates
     └→ docs/PROJECT_MASTER_HANDOFF.md updated with completion

  2. Submit task via Slack
     └→ Task Router receives via Zapier
     └→ Routes to appropriate agent
     └→ Tracks in Google Sheets
     └→ Agent completes, sends callback
     └→ Status updates, Slack notification sent

TROUBLESHOOTING
  Port conflict?            → Change port in .env
  Webhook not firing?       → Check .env webhook URLs
  Agent not receiving?      → Verify Zapier Zap is active
  Logs not appearing?       → Check logs/ directory exists
  Service won't start?      → npm install, check Node version

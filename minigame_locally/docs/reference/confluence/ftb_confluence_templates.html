Feed the Bear — Confluence Templates

Two reusable templates for Confluence: a User Macro for styled sections, and a full Page Template for new design docs.

Option A: User Macro (Admin required)

What is it? A User Macro is a reusable widget you insert into any Confluence page with {ftb-section}. It wraps your content in the FTB visual style (white card, green underline, correct colors).

Requirement: Confluence Server/Data Center with admin access. On Confluence Cloud, use the Page Template instead (Option B).

1Go to Admin → User Macros → Create

Option B: Page Template (No admin required)

What is it? A page template that any space admin can add. When someone creates a new page, they pick "FTB Design Doc" from the template list and get the full structure pre-filled.

Works on: Confluence Cloud and Server/Data Center.

1Go to Space Settings → Content Tools → Templates → Create Template

2Name it: FTB Design Doc

3Switch to the source editor (< > icon) and paste this:

Confluence Cloud Note

User Macros (ftb-section, ftb-header, ftb-table) are not available on Confluence Cloud. On Cloud, use the Page Templates (Option B) or paste the full HTML into an HTML macro on each page. If your team is on Cloud and wants reusable macros, you'd need to build a Forge app — which is a bigger lift. The Page Templates are the practical choice for Cloud.
